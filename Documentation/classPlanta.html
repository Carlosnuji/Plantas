Descripción detallada

Clase para el manejo de las plantas de la base de datos.

Con esta clase se podrá crear, eliminar, cargar, actualizar y buscar plantas en la base de datos. A su vez se podrá crear un JSON de la clase.

int main()
{
Planta planta("Nombre", "Nombre científico", "Descripción");
planta.save();
planta.load(2);
planta.toJSON();
planta.remove();
Planta::find("Nombre planta")
}
