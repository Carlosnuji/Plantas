Descripción detallada

Clase para el manejo de la conexión con la base de datos.

Se podrá abrir la conexión con la base de datos a partir de la lectura de un archivo de configuración y guardara sus propiedades. Se necesita los includes de qtsql para poder hacer la conexión.

int main()
{
Conexion conexion();
conexion.open();
conexion.close();
}
