Descripción detallada

Clase para el manejo de los favoritos de los usuarios.

Se podrá crear e insertar un nuevo favorito en la base de datos y eliminarlo, comprobar si esa planta la tiene el usuario en favorito o no y buscar todas las plantas que tiene en favoritos un usuario.

int main()
{
Favorito favorito(1, 1);
favorito.load();
bool b = favorito.check();
favorito.find(1);
}

◆ check()

bool Favorito::check ( )

Comprueba si esa planta la tiene en favoritos el usuario.

Hace un select haciendo un count y si devuelve 1 o más la tiene en favorito. Si la tiene en favorito lo eliminamos y si no la tiene la añadimos a favorito.

Devuelve
True si la tiene en favorito y false si no la tiene en favorito.

return 1 si existe return 0 si no existe
