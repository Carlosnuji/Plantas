Clase para poder crear Emails.

Sirve para poder enviar emails via smpt gracias a la libreria SimpleMail descargada desde GitHub.

{
Email email();
email.sendEmail("toTest@gmail.com", "Subject", "Text");

}

void Email::sendEmail ( std::string  toEmail,
std::string  subject,
std::string  texto 
)

Poder enviar un email via smtp.

Se crea un objeto Sender de la libreria SimpleMail y se cargar las propiedades necesarias como, el usuario, la contraseña, el servidor y el envio mediante SSL. Se añaden la dirección, el asunto y el texto y se envia el email.

Parámetros
toEmail- Dirección a la que se va a enviar el email.
subject- El asunto del email.
texto- Texto que se enviará en el email.
