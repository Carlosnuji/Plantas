Descripción detallada

Clase para el manejo de las quejas en la base de datos.

Con esta clase de podrá crear y elminar una queja, cargar una queja de un usuario, obtener una lista de todas las quejas de un usuario y crear un JSON de la clase para enviar al cliente.

int main()
{
Queja queja(1, "Queja");
queja.save();
queja.load(1);
Queja::remove(1);
Queja::find(1);
}
