Descripción detallada

Clase para el manejo de los usuarios de la base de datos.

Con esta clase se podrá crear, eliminar, cargar, actualizar y buscar usuarios en la base de datos. A su vez se podrá crear un JSON de la clase.

int main()
{
Usuario usuario("name", "pass", "email");
usuario.save();
usuario.load(1);
usuario.update("nombre", "email", 1);
usuario.toJSON();
Usuario::remove(1);
}
