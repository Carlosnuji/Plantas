Descripción detallada

Clase para el manejo y generación de tokens.

Con esta clase se podrá crear, insertar en la base de datos o comprobar si un token existe en la base de datos.

int main()
{
Token token();
QString tokenString = token.generateToken();
token.insert(tokenString, 1);
Token::checkToken(tokenString);
}
